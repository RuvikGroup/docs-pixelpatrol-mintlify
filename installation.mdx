## Overview

Pixel Patrol can be integrated into any application through our REST API. This guide covers installation options, authentication setup, and basic configuration.

## Integration Methods

### REST API

The primary integration method for all platforms:

<CodeGroup>

## Basic Configuration

### Initialize Client

Create a reusable client configuration:

<CodeGroup>
```javascript JavaScript
class PixelPatrolClient {
  constructor(apiKey, options = {}) {
    this.apiKey = apiKey;
    this.baseUrl = options.baseUrl || 'https://api.pixelpatrol.app/v1';
    this.timeout = options.timeout || 30000;
  }

  async submitMedia(mediaData) {
    const response = await fetch(`${this.baseUrl}/media/submit`, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${this.apiKey}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(mediaData),
      timeout: this.timeout
    });
    
    if (!response.ok) {
      throw new Error(`API error: ${response.status}`);
    }
    
    return response.json();
  }
}

const client = new PixelPatrolClient(process.env.PIXELPATROL_API_KEY);
```

```python Python
import requests
from typing import Optional, Dict, Any

class PixelPatrolClient:
    def __init__(self, api_key: str, base_url: Optional[str] = None, timeout: int = 30):
        self.api_key = api_key
        self.base_url = base_url or 'https://api.pixelpatrol.app/v1'
        self.timeout = timeout
        self.session = requests.Session()
        self.session.headers.update({
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        })
    
    def submit_media(self, media_data: Dict[str, Any]) -> Dict[str, Any]:
        response = self.session.post(
            f'{self.base_url}/media/submit',
            json=media_data,
            timeout=self.timeout
        )
        response.raise_for_status()
        return response.json()

client = PixelPatrolClient(os.environ['PIXELPATROL_API_KEY'])
```
</CodeGroup>

## Framework Integration

### Next.js

```javascript
// app/api/moderate/route.js
import { NextResponse } from 'next/server';

export async function POST(request) {
  const body = await request.json();
  
  const response = await fetch('https://api.pixelpatrol.app/v1/media/submit', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.PIXELPATROL_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      url: body.imageUrl,
      type: 'image',
      metadata: body.metadata
    })
  });
  
  const result = await response.json();
  return NextResponse.json(result);
}
```

### Express.js

```javascript

const app = express();

app.post('/moderate', async (req, res) => {
```

## Error Handling

### Common Errors

Implement proper error handling:

```javascript
try {
  const result = await client.submitMedia(mediaData);
} catch (error) {
  if (error.status === 401) {
    console.error('Invalid API key');
  } else if (error.status === 429) {
    console.error('Rate limit exceeded');
  } else if (error.status === 400) {
    console.error('Invalid request:', error.message);
  } else {
    console.error('Unexpected error:', error);
  }
}
```

### Retry Logic

Implement exponential backoff:

```javascript
async function submitWithRetry(mediaData, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await client.submitMedia(mediaData);
    } catch (error) {
      if (error.status === 429 || error.status >= 500) {
        const delay = Math.pow(2, i) * 1000;
        await new Promise(resolve => setTimeout(resolve, delay));
      } else {
        throw error;
      }
    }
  }
  throw new Error('Max retries exceeded');
}
```
